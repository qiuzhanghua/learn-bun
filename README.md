## init a .env file for bun

```env

ACTIVE_DB=mysql # mysql, pg, sqlite, mssql

SQLITE_PATH=memory
# SQLITE_PATH=/tmp/bun.db

MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=app
MYSQL_PASSWORD=app
MYSQL_DATABASE=app

PG_HOST=localhost
PG_PORT=5432
PG_USER=app
PG_PASSWORD=app
PG_DATABASE=app

MSSQL_HOST=localhost
MSSQL_PORT=1433
MSSQL_USER=app
MSSQL_PASSWORD=app
MSSQL_DATABASE=app
```
